# LearnSphere LMS V2

Final course-based Learning Management System.

## Features
- Professional landing page
- Course search and category filter
- Enrollment and course details
- Full course learning screen
- 10 lessons per course
- Lesson notes and practical examples
- Previous/Next lesson navigation
- Mark Lesson Complete
- Dynamic progress stored in localStorage
- Assignment submission
- Video lesson popup
- Login popup
- Dark/light mode
- Certificate at 100%
- Responsive design

## Run
npm install
npm run dev
